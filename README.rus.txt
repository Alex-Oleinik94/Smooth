Привет! Это README к SaGe. 
Тут маломальски собрана вся информация, которая поможет вам разобраться и использовать это чудо.

Чтобы билдить нужно копировать скрипты из Build\Platforms\{TARGET_PLATFORM}\Scripts в 
	Build\Scripts и их можно использовать оттуда. 
Для первой компиляции используются специальные скрипты. Это сделано потому что для многих функций
	(Например билдить файлы в юниты) при компиляции используется исполняемый файл движка.
Отдельно из движка в папки Build\Packages\{PACKAGE_NAME} вынесены пакеты всяких прикалюх, 
	для компиляции пакетов в испольняемый файл так-же есть специальные скрипты.

Чтобы билдить под Android:
	C:\Programming\android-ndk   --- Сюда устанавливаем Android Native Development Kit 
	C:\Programming\android-sdk   --- Сюда устанавливаем Android SDK
	C:\Programming\apache-ant    --- Сюда устанавливаем Apache Ant
	C:\Programming\jdk           --- Сюда устанавливаем Java Development Kit

Дополнительно:
	Alt + Enter    --- Полноэкранный режим
	Ctrl + Alt + O --- Панель конфигурации

Из интересного:
	Движок поддерживает RUNTIME смену рендера.
	В пакете фракталов есть красивые прикалюхи.
	Чтобы приближаль во фрактале Мандельброта сначала кликаем правой клавишей мышки,
		потом выбераем боласть и затем для отрисовки кликаем левой клавишей.
